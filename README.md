# SE Competency Tracker

A comprehensive web application for tracking Sales Engineer competencies across 6 key categories with 22 total competencies.

## Features

- **Manager View**: Rate team members on competencies with detailed descriptions for each level
- **Employee View**: Self-assessment capabilities for employees
- **Dashboard**: Visual analytics with charts and gap analysis
- **Team Management**: Add/remove team members
- **Notes**: Manager and employee notes for each competency

## Usage

1. Open the application in your web browser
2. Add team members in the Manager View
3. Select a team member and begin rating their competencies
4. Switch to Employee View for self-assessments
5. View the Dashboard for visual analytics and progress tracking

## Data Storage

All data is stored locally in your browser's localStorage. Each user's data is stored separately on their device.

## Categories

3. Team Selling
